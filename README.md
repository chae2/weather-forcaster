## 사용 스택
- Framework: Next.js (React)

- API Interface: GraphQL (Apollo Server Micro)

- Data Source: OpenWeatherMap API

- Styling: CSS Modules

- Font: Pretendard

## Prerequisites
Prerequisites
프로젝트 실행을 위해 다음 사항들이 필요합니다.

Node.js (v14 이상 권장)

NPM 또는 Yarn

OpenWeatherMap API Key (발급 받기)

## 설정 및 실행방법
1.`npm install`
2. 환경변수에 `OPENWEATHER_API_KEY` 추가
3. `npm run dev`로 실행
